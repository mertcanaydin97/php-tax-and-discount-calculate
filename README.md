## Usage | Kullanım

For function detail look for functionalized.php

Fonksiyon detayları için functionalized.php dosyasına bakın.

/* Tax or Discount etc*/

echo percent_calculate(10000, 20, 'result'); /* Equals | Eşittir 12000 */
